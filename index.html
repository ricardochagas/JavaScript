Aula 1 - Variáveis

Tipos de variáveis:

  • number:
    integer, float
  • string:
    conjunto de caracteres dentro de aspas simples '' ou aspas duplas ""
  • boolean:
    true ou false
  • null:
    não existe e não tem valor
  • undefined:
    existe, mas ainda não tem valor definido

Notação para uso de variável:

Aula 3 - Operadores de

Operadores de Igualdade

  • == Igual a
    Compara valores e retorna true ou false
    var x = 0
    undefined
    x
    0
    x == 0
    true
    x == 5
    false
    var nome = "joao"
    undefined
    nome
    "joao"
    nome == "joao"
    true
  • ATENÇÃO: o JS tenta converter valores para verificar se o valor testado é igual ao outro.
    Isto se aplica a um valor string testado e comparado a um valor number, por exemplo. Isto acontece devido ao JS ser uma linguagem com tipagem dinâmica.
    var um = "1"
    um
    "1"
    var numum = 1
    numum
    1
    um == numum
    true
  • != Diferente de
    A mesma notação de igualdade serve para a diferença
  • === Igual a, e testa se é do mesmo tipo, ou seja, (valor = valor e number = number ou string = string)
    um === numum
    false
  • !== Diferente, mas testa se é do mesmo tipo. A mesma notação de igualdade serve para a diferença
    um !== numum
    true

Operadores Relacionais

Devemos considerar a variável num.
var num = 100
undefined
num
100

  • Maior que >
    num > 10
    true
  • Menor que <
    num < 5
    false
  • Marior ou igual a >=
    num >= 200
    false
  • Menor ou igual a <=
    num <= 100
    true
